# Button & LED RGB

Este es un programa que permite interactuar con un pulsador y un LED RGB para que cambie de color cada vez que es presionado. Por ejemplo, al encender el MCU est√° el led apagado, si presiono el pulsador por primera vez se pone de color rojo, al presionarlo por segunda vez se pone de color verde, al presionarlo por tercera vez se pone de color azul, y al presionarlo por cuarta vez se termina el ciclo apagando el LED permitiendo empezar de nuevo.
